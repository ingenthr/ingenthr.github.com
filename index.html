Source from NorthScale

NorthScale builds upon other great Open Source projects and releases Open Source to the community in the process. This site lists projects which NorthScale has created or made contributions to. NorthScale believes in "standing on the shoulders of giants" in building upon and contributing to Open Source.

NorthScale has released moxi (a memcached proxy) and developers from NorthScale have contributed to memcached, spymemcached, beanstalkd, git and others.

moxi

Moxi is a memcached proxy with several features that can help keep the memcached contract whole in complicated environments. It brings several optimizations to memcached deployments without requiring any changes to the application software using memcached.

Dependencies

  • libmemcached (specific version, statically linked and bundled)
  • libevent
    • , a portable event notification layer for socket/file descriptor events on UNIX like systems
  • check, a C unit test framework (needed only for testing, we're fans of test driven design)

For more information on depedencies, see the project wiki. Documentation for building is a bit rough at the moment, but will be cleaned up soon.

Installation

Packages will shortly be available for the most common platforms, and that is the preferred method of installation. Otherwise, the source is automake/autconf based and should build on most UNIX like systems.
