Source from NorthScale

NorthScale builds upon other great Open Source projects and releases Open Source to the community in the process. This site lists projects which NorthScale has created or made contributions to. NorthScale believes in "standing on the shoulders of giants" in building upon and contributing to Open Source.

NorthScale has released moxi (a memcached proxy) and libstrophe. Developers from NorthScale have contributed to memcached, spymemcached, beanstalkd, strophe, git and others.

moxi

moxi is a memcached proxy with several features that can help keep the memcached contract whole in complicated environments. It brings several optimizations to memcached deployments without requiring any changes to the application software using memcached.

libconflate

libconflate is a generalized XMPP agent designed to embed in a C applciation to enable remote XMPP configuration and control of an appliction.
